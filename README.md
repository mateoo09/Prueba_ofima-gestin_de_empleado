# Sistema de Gestión de Empleados

Aplicación web desarrollada para la gestión de empleados dentro de una organización.  
El sistema permite registrar usuarios, iniciar sesión y administrar la información de empleados mediante una arquitectura separada de **Backend API REST** y **Frontend Web**.

---

# Tecnologías utilizadas

## Backend

El backend fue desarrollado con tecnologías del ecosistema .NET para exponer una API REST.

Tecnologías:

- C#
- ASP.NET Core Web API
- Entity Framework
- SQL Server
- JWT (JSON Web Token) para autenticación

Funciones principales:

- Registro de usuarios
- Inicio de sesión
- Autenticación con JWT
- CRUD de empleados
- Conexión con base de datos

---

## Frontend

El frontend corresponde a la interfaz gráfica donde el usuario interactúa con el sistema.

Tecnologías:

- Angular
- TypeScript
- HTML
- CSS

Funciones:

- Interfaz de inicio de sesión
- Registro de usuarios
- Visualización de empleados
- Gestión de empleados
- Consumo de la API REST
